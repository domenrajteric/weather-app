Vremenska aplikacija (Flask + OpenWeather API).

Preprosta spletna aplikacija, izdelana v Pythonu z uporabo ogrodja Flask, ki omogoča prikaz trenutnega vremena za izbrano mesto s pomočjo OpenWeather API-ja.

1. Namesti potrebne knjižnice: "pip install -r req.txt"
2. Ustvari brezplačen API ključ na OpenWeather: https://openweathermap.org/

API ključ mora ostati skrit, kar pomeni, da ga v testnem okolju lahko "ročno" zapišeš v "main.py" datoteko, vendar ga moraš v nadaljevanju,
ko projekt dodaš na GitHub (in Render, npr.), zamenjati z okoljsko spremenljivko, ki je zapisana v sistem in na tak način tudi zavarovana.
Na spodnji sliki je primer iz Render.com, kjer tik pred objavo, dodaš vse okoljske spremenljivke, v tem primeru je to "API_KEY" in tvoj API ključ.
